1. Open BookBrowserPrototype.java.
2. Go to 50th line, the bookTitle arrary. Update/Add the full names of
   books (which are shown in the drop down menu) in order.
3. Go to 55th line, the photoName arrary. Update/Add the short names of
   books (which are included in the photos of this book) in order.
4. Go to 58th line, the totalPage arrary. Update/Add the total pages of
   books (how many photos will be included in this book) in order.
Note: The data in each arrary correspond to each other.
      i.e. The 1st element in each arrary represents the information
      of the first book imported, the 2nd element in each arrary represents
      the information of the second book imported...
